  3. “03 数字滤波器”3-4算法结构中，IIR滤波器的直接型设计均在a前漏了“-”号。注意分母上是有负号的则滤波器中不再出现。

  4. “03 数字滤波器”3-1数字滤波器基础中，数字滤波器的参数图疑似有误，应该为下图，且通带最大衰减和阻带最小衰减应当再取-20log转化为dB值。

  5. 【补】IIR滤波器与FIR滤波器的差异

    • FIR滤波器单位脉冲响应有限长，IIR无限长；
    • FIR滤波器极点固定在原点，IIR极点位于单位圆内任何地方；
    • FIR阶次高，IIR阶次低；
    • FIR可实现严格线性相位，IIR不一定；
    • IIR采用递归结构，FIR一般采用非递归结构；
    • FIR可用FFT计算，IIR不能用FFT计算；
    • IIR可用模拟滤波器设计，FIR借助计算机设计

  6. 【补】DTFT、Z变换、DFT关系
